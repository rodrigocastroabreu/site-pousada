# Pousada Secreta

Site estático da Pousada Secreta, com página inicial e página de quartos.

## Estrutura do projeto

- index.html: página principal com apresentação, rota, sobre a pousada e contato.
- quartos.html: página com os tipos de quartos disponíveis.
- css/: arquivos de estilos do site.
  - style.css: estilos da página inicial.
  - estilo-quarto.css: estilos da página de quartos.
- assets/: imagens e ícones usados no projeto.
  - images/: fotos da pousada e dos quartos.
  - icons/: ícones utilizados no rodapé.

## Como visualizar

1. Abra o arquivo index.html no navegador.
2. Ou use um servidor local simples, como o Live Server no VS Code.

## Sobre o projeto

Este projeto é uma landing page estática para uma pousada, com foco em apresentação visual, navegação entre páginas e organização de arquivos.
